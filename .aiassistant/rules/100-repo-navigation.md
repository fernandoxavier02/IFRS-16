# Repo Navigation

Guide to finding things in this repository:

- **Frontend Pages**: Root directory (`*.html`).
- **Frontend Logic**: `assets/js/` (calculator, auth, stripe).
- **Backend Entry Point**: `backend/app/main.py`.
- **Backend Models**: `backend/app/models.py`.
- **Backend Business Logic**: `backend/app/services/` and `backend/app/routers/`.
- **Database Migrations**: `backend/alembic/versions/`.
- **Tests**: `backend/tests/`.
- **AI Context**: `docs/ai/`.
- **MCP Integrations**: `mcp/`.
